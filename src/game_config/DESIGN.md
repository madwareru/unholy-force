# Высокоуровневое описание системы конфигов
Есть несколько основных типов конфигурационных файлов:
+ Описание основных настроек игры (GameConfig)
+ Описания персонажей (UnitConfig)
+ Описания предметов (ItemConfig)
+ Части, из которых собираются этажи (FloorPartConfig)
+ Описание правил смежности частей этажей (FloorPartAdjacencyConfig)
+ Описания этажей (FloorConfig)
+ Описания связей между этажами (FloorFlowGraphConfig)
+ Описания игровых параметров (ParameterConfig)
+ Описания игровых тегов (TagConfig)
+ Описания эффектов для игровых механик (EffectMechanicConfig)
+ (Не в первой версии) Описания эффектов для разных поведений ИИ
+ (Не в первой версии) Описания эффектов диалогов для персонажей с заданиями и торговцев
